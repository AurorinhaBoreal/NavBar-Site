CSS - Cascading Style Sheet

O CSS como o próprio nome ja diz, é uma linguagem de estilização. Ele é usado com outras linguagens de marcação (como o HTML ou XHTML) para modificar o aspecto da página e deixar ela mais bonita. Ele permite desde criar tabelas, menus expansiveis, até mesmo diversas animações e layouts de calculadora por exemplo.
O CSS foi criado pelo World Wide Web Consortium - W3C. Inicialmente houve uma tentativa de implementar a estilização, mas o HTML simplesmente não tinha esse próposito. A linguagem não tinha sido feita para utilização de tags, e como cada site tinha suas próprias fontes e palheta de cores, ficava muito caro e trabalhosa a reescrita dos códigos do zero para a implementação de estilos utilizando HTML.
É possivel a criação de sites em html sem utilizar o CSS, porém fica com um aspecto muito grosseiro e não é nem um pouco atraente pro usuário médio.
